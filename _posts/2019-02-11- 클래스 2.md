# 백지노트

## Getter & Setter
 
>최상위 수준 함수 : 클래스 외부에 정의한 함수 소스 코드 파일(.kt)에 바로 정의하여 사용한다는 의미. 전체적으로 사용하는 기능을 처리할 때 유용하다. ~Java의 private static final~

>최상위 수준 속성 : 클래스 외부에 선언. 최상위 수준 변수와 클래스 내부에 정의된 멤버 변수를 **속성**이라고 표현한다.


코틀린에서의 속성 : 최상위 변수, 클래스 멤버 변수
코틀린에서의 필드 : 함수 지역 변수

속성이 만들어 지면 getter와 setter가 자동으로 생성된다.
